\doxysubsection{Detailed Description}
Sensor data structure with mutex protection. 

Contains sensor readings from accelerometer, gyroscope, magnetometer, and ADCs. \begin{DoxyPrecond}{Precondition}
Acquire mutex before performing any read/write on this structure.
